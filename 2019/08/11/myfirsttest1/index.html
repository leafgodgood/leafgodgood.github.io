本文关键字： 数组，二维数组，for循环，空指针nullptr，while循环

题一：
面试题3（一）：找出数组中重复的数字
题目：在一个长度为n的数组里的所有数字都在0到n-1的范围内。数组中某些数字是重复的，但不知道有几个数字重复了，
也不知道每个数字重复了几次。请找出数组中任意一个重复的数字。例如，如果输入长度为7的数组{2, 3, 1, 0, 2, 5, 3}，
那么对应的输出是重复的数字2或者3。

思路一：取索引为0的数，与其他数一一对比，取索引为1的数，与其他一一对比，输出打印，用两个for循环和一个if，注意内置for循环索引大于外置for循环索引。

思路二：

思路一代码：

‘’’

#include
//找出长度为n的数组里包含一个或多个重复的数字
using namespace std;
int FindLikeArray(int numbers[],int n) {
int j = 1;
for (int i = 0; i < n; ++i) {

    for (int j=i+1; j < n; j++) {
        if (numbers[i] == numbers[j]) {
            cout << numbers[j] ;
            //cout << numbers[j] << endl;
        }            
    }
}
    return 0;

}
void test() {
int numbers[] = {6,1,2,2,6,1,3 };
FindLikeArray( numbers,7);
}

int main() {
test();
}
